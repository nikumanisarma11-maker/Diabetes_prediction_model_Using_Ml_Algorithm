# Diabetes Prediction using Machine Learning

## Project Overview
This project predicts whether a patient has diabetes using Machine Learning algorithms.

## Dataset
- diabetes.csv
- Features include:
  - Pregnancies
  - Glucose
  - Blood Pressure
  - Skin Thickness
  - Insulin
  - BMI
  - Diabetes Pedigree Function
  - Age

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning
The model is trained to classify whether a patient is diabetic or non-diabetic.
